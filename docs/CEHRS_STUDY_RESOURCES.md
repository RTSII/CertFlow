# CEHRS Study Resources Reference

> This document contains the comprehensive study resource list for the CEHRS certification exam.

---

## Study Schedule Overview

| Weeks | Focus Area |
|-------|------------|
| Weeks 1-2 | Non-Clinical Operations + EHR fundamentals |
| Weeks 3-4 | Clinical Operations |
| Week 5 | Revenue Cycle / Finance |
| Week 6 | Regulatory Compliance + Reporting |
| Weeks 7-8 | Practice tests + weak-area review |

---

## Category: Official & High-Yield (Start Here)

### 1. NHA CEHRS Test Plan (Blueprint)
- **Link:** https://www.nhanow.com/docs/default-source/test-plans/nha_cehrs-test-plan.pdf
- Best single document for what's on the exam
- Lists each domain, task statements, and knowledge statements
- Use it as your master checklist

### 2. NHA Candidate Handbook
- **HTML:** https://knowledge.nhanow.com/nha-candidate-handbook
- **PDF:** https://www.nhanow.com/docs/default-source/test-plans/candidate_handbook.pdf
- Explains eligibility, registration, scoring, retakes, and policies
- Read once to avoid surprises (especially rules for ID, rescheduling, misconduct, etc.)

### 3. NHA CEHRS Exam Page
- **Link:** https://www.nhanow.com/certification/nha-certifications/certified-electronic-health-records-specialist-%28cehrs%29
- Short "5 steps to certification" overview
- Confirms eligibility, test time, fees

### 4. NHA Certification Portal
- **Link:** https://certportal.nhanow.com/
